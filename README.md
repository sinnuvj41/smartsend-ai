# 🚀 SmartSend AI

Advanced Contact Form Script with AI + Dashboard + Real-time Notifications

---

## ✨ Features

- 🤖 AI message improvement (Google Gemini)
- 📩 Contact form with file upload
- 📧 Email notifications (SMTP)
- 🔐 Admin dashboard with login
- 📊 Analytics charts
- 🔔 Real-time notifications (Socket.io)
- 📁 File download support
- 📤 CSV export
- 📱 Fully responsive UI
- 🟢 Online users indicator

---

## 🛠 Tech Stack

- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Database: MongoDB
- Real-time: Socket.io

---
