SmartSend AI Documentation

SmartSend AI is a modern contact form application with AI message improvement, file uploads, email notifications, admin dashboard, analytics charts, real-time notifications and submission management.

Features

  • AI-powered message improvement using Google Gemini API
  • Advanced contact form with file uploader
  • Email notifications using SMTP / Nodemailer
  • User confirmation email
  • Admin login with JWT authentication
  • Submission management dashboard
  • View, delete, read/unread and archive submissions
  • CSV export
  • Analytics charts
  • Real-time notifications using Socket.io
  • Online users indicator
  • Responsive premium UI

Technical Requirements

  • Node.js 18 or higher
  • MongoDB Atlas or local MongoDB
  • NPM
  • Modern browser
  • SMTP email account
  • Google Gemini API key for AI feature
External services like MongoDB Atlas, Google Gemini API and SMTP email providers may require separate accounts and may have usage limits or costs.

Installation

Extract the downloaded package and open the project folder.

smartsend-ai/
├── client/
├── server/
├── documentation/
└── README.md

Backend Setup

cd server
npm install

Create a .env file from .env.example.

copy .env.example .env

Start backend:

npm run server

Backend default URL:

http://localhost:5000

Frontend Setup

cd client
npm install
npm run dev

Frontend default URL:

http://localhost:5173

Environment Variables

PORT=5000
NODE_ENV=development

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
JWT_EXPIRES_IN=7d

ADMIN_EMAIL=admin@example.com
ADMIN_PASSWORD=your_admin_password

CLIENT_URL=http://localhost:5173

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your_email@gmail.com
SMTP_PASS=your_app_password
MAIL_FROM_NAME=SmartSend AI
MAIL_FROM_EMAIL=your_email@gmail.com
ADMIN_RECEIVER_EMAIL=your_email@gmail.com

GEMINI_API_KEY=your_gemini_api_key

MAX_FILE_SIZE_MB=5
UPLOAD_DIR=uploads
Never share your real .env file publicly. Only include .env.example in your final package.

SMTP Setup

For Gmail SMTP, enable 2-Step Verification and create an App Password.

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your_email@gmail.com
SMTP_PASS=your_16_character_app_password

Google Gemini AI Setup

Create a Gemini API key from Google AI Studio and add it to backend .env:

GEMINI_API_KEY=your_gemini_api_key

Admin Dashboard

Admin login URL:

http://localhost:5173/admin

Default credentials are configured in the backend .env file:

ADMIN_EMAIL=admin@example.com
ADMIN_PASSWORD=your_admin_password

Deployment

Backend Deployment

Deploy the server folder to services like Render, Railway or VPS.

Set all backend environment variables in your hosting dashboard.

Frontend Deployment

Update client environment variables:

VITE_API_URL=https://your-backend-url.com/api
VITE_SOCKET_URL=https://your-backend-url.com

Build frontend:

cd client
npm run build

Upload the dist folder to Netlify, Vercel or your hosting provider.

Support Notes

  • Make sure MongoDB connection string is correct.
  • Use Gmail App Password, not normal Gmail password.
  • Check backend logs if email or AI feature fails.
  • For production, update CLIENT_URL and frontend API URLs.
